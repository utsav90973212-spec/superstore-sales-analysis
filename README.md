# 🛒 Superstore Sales Intelligence Dashboard

## Project Overview
A complete Business Analytics project analyzing 10,194 sales 
transactions from a US Superstore (2023–2026) using Python 
and Power BI.

## Business Questions Answered
1. Does discounting hurt profit?
2. Which region performs best?
3. Which product category is most profitable?
4. How are sales trending over time?

## Key Findings
- Discounts above 30% consistently generate losses
- West region leads with $111K profit vs Central at $40K
- Technology earns 7x more profit than Furniture
- Sales grew 5x over 4 years with clear holiday seasonality

## Recommendations
- Cap maximum discount at 25%
- Introduce loyalty programs in Central region
- Invest more in Technology category
- Run January clearance sales to recover slow month revenue
- Stock up before November for holiday season demand

## Tools Used
- Python (pandas, matplotlib, seaborn, plotly)
- Power BI Desktop
- Jupyter Notebook / Google Colab
